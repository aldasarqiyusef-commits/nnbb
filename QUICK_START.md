# Quick Start Guide - دليل البدء السريع

### الخطوة 1: التحميل
```
اذهب إلى: https://github.com/aldasarqiyusef-commits/nnbb/releases
حمّل أحدث نسخة
```

### الخطوة 2: استخراج الملفات
```
افتح الملف المضغوط
استخرج جميع الملفات في مجلد
```

### الخطوة 3: تشغيل اللعبة
```
شغّل ملف .exe
استمتع باللعبة!
```

---

## 🛠️ للمطورين

### الخطوة 1: تحميل Unity
```
https://unity.com/download
حمّل Unity 2020.3 LTS أو أحدث
```

### الخطوة 2: استنساخ المستودع
```bash
git clone https://github.com/aldasarqiyusef-commits/nnbb.git
```

### الخطوة 3: فتح في Unity
```
افتح Unity Hub
اختر "Open"
اختر مجلد المشروع
```

### الخطوة 4: تشغيل اللعبة
```
اضغط Play ▶️ في Unity Editor
```

### الخطوة 5: البناء للتوزيع
```
File → Build Settings
اختر النظام المستهدف (Windows/Mac/Linux)
اضغط Build
```
