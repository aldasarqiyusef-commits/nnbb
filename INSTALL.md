# Installation Guide - دليل التثبيت

## English

### For Players (Non-Developers)

#### Method 1: Download Executable (When Available)
1. Go to Releases: https://github.com/aldasarqiyusef-commits/nnbb/releases
2. Download the latest Windows build (.zip)
3. Extract the folder
4. Run the `.exe` file
5. Play!

#### Method 2: Play in Web Browser
When WebGL build is available, play directly in your browser (no installation needed).

## Troubleshooting

### Game won't start?
1. Check if you extracted all files
2. Ensure _Data folder is in the same directory as .exe
3. Update your graphics drivers
4. Disable antivirus temporarily (false positive)

### Game is slow?
1. Close background applications
2. Lower in-game graphics quality (if available)
3. Check system requirements

### Game crashes?
1. Update graphics drivers
2. Verify you have .NET Framework installed
3. Run as Administrator
